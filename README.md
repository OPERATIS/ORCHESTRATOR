1. ```copy .env.example .env```
2. ```docker-compose up -d```
3. Go to docker\
   ```docker-compose exec app bash```
4. Run inside docker\
   ```composer install```
5. Exit from docker\
   ```exit```
6. Run from local environment\
   ```npm i```\
   ```npm run dev```
7. Configuration local environment\
   ```sudo nano /etc/hosts```\
   127.0.0.1 connect-loc.com\
   127.0.0.1 socket.connect-loc.com
